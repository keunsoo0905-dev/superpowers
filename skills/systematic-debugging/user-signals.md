# User Signals: You're Doing It Wrong

Watch for these redirections from your human partner. Each signals a specific process failure.

## Signal-to-Action Map

| Signal | What It Means | Required Action |
|---|---|---|
| "Is that not happening?" | You assumed without verifying | Return to Phase 1 -- add verification |
| "Will it show us...?" | You should have added evidence gathering | Add diagnostic instrumentation |
| "Stop guessing" | You're proposing fixes without understanding | STOP. Full Phase 1 restart |
| "Ultrathink this" | You're treating symptoms, not questioning fundamentals | Step back, question architecture |
| "We're stuck?" (frustrated) | Your approach isn't working | Acknowledge, change strategy entirely |

## General Rule

**When you see ANY of these signals:** STOP immediately. Return to Phase 1. Do not continue the current approach.

These signals indicate your human partner has recognized a process violation before you did. Trust their judgment.
